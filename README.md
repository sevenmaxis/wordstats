# Wordstats

Steal word statistics from wordstat.yandex.ru

## Installation

Add this line to your application's Gemfile:

    gem 'wordstats'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install wordstats

## Usage

Type command `wordstats`, wait for some time
